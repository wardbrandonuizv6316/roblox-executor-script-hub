## Roblox Executor Overview

Roblox Executor is a Windows desktop application for opening and running Roblox Lua and Luau scripts through a straightforward interface. Its design combines script execution and script hub access in a single utility, keeping the workflow focused rather than adding unrelated setup components.

The application is intended for 64-bit Windows 10 and Windows 11 computers. Its core workflow includes rapid injection, drag-and-drop file loading, and real-time execution feedback. A built-in hub also provides a central place to browse available scripts and start them.
